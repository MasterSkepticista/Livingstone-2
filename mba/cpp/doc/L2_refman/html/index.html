Livingstone Reference Manual

 

The Livingstone Reference Manual is not included in the Livingstone distribution. You can generate the Reference Manual automatically from the Livingstone source code using the open-source Doxygen tool. The instructions for doing so are in a file called doxygen.html. The configuration file supplied with the Livingstone distribution will place the Livingstone Reference Manual's home page at the URL of the present page.

The latest version of the Livingstone Reference Manual can be found on the World-Wide Web at http://ic.arc.nasa.gov/projects/mba/projects/L2/doc/L2_refman/html/index.html .
