NASA export laws prevent me from providing a link to download the Oliver application. Oliver is packaged with Livingstone(L2) and is only available by means of a Software Usage Agreement. Contact Will Taylor for more information.

Before you can run Oliver you must correctly install Livingstone on your system. See Building the Livingstone2 Engine for more detailed instructions.

Most people who read this will probably be experienced Livingstone users, and already have the engine installed on there system. To run Oliver you only need the jar file located at /mba/skunkworks/groundworks/jars/oliverfull.jar, and java jdk1.4 installed on your system. The Java virtual engine is free and can be download from http://www.javasoft.com.

To start the Oliver editor, see Starting Oliver (Stanley II)

At this time I have not written detailed documentation, but if you need more instruction please contact me and I will be happy to help you get started. More user documentation will follow now that most of the code has been written and I have more time.
