Running Livingstone/Oliver (Stanley II) on Ames Solaris Platforms




# Solaris 5.8

In a fresh xterm:
# set your JAVA_HOME, e.g.
% setenv JAVA_HOME /usr/local/j2sdk1.4.1

% cd /home/taylor/L2Skunk/released_2.7.8.1/skunkworks/groundworks
% ./oliver



Running Livingstone/Oliver (Stanley II) on Solaris or Linux Platforms




In a fresh xterm:
# set your JAVA_HOME, e.g.
% setenv JAVA_HOME /home/wtaylor/pub/jdk1.4.1_01

% cd <root_directory>/skunkworks/groundworks
% ./oliver



Running Livingstone/Oliver (Stanley II) on Windows Platforms




In Cygwin window
----------------
% tcsh
# set your JAVA_HOME, e.g.
% setenv JAVA_HOME D:/progra~1/jdk1.4.1_01

% cd D:/cygwin/home/wtaylor/L2Skunk/skunkworks/groundworks
% ./oliver -win32



In Command Prompt window
------------------------
# set your JAVA_HOME, e.g.
% set JAVA_HOME=D:/progra~1/jdk1.4.1_01

% cd <root_directory>
% %JAVA_HOME%\bin\java -jar skunkworks\groundworks\jars\oliverfull.jar
